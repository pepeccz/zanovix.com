# zanovix.com — Contexto de proyecto

Web de captación de leads de Zanovix (IA aplicada a pyme). Astro 6 SSR + React
islands + Tailwind 4 (`@theme` en `src/styles/global.css`) + GSAP + Lenis.

## Design Context (fuente de verdad — leer antes de tocar diseño o copy)

- **`PRODUCT.md`** (raíz) — estrategia: registro `brand`, comprador dual, propósito,
  personalidad, anti-referencias, 5 principios de diseño, accesibilidad.
- **`DESIGN.md`** (raíz) — sistema visual: paleta (un acento), 4 familias
  tipográficas con roles, escala fluida, motion, componentes, Do's/Don'ts.
- **`docs/estrategia/01-estrategia-mensaje.md`** — estrategia de mensaje firmada:
  posicionamiento (Puente), 3 dolores, mensaje central, voz, mapeo sección→mensaje.

## Reglas no negociables

- **Idioma:** castellano peninsular, trato de "tú", **sin voseo**. Sin guiones largos (—).
- **Voz:** anti-hype. Sin "transformación digital / disruptivo / 10x / agencia /
  consultora". Fórmula **jubilada**: "No vendemos X. Devolvemos Y." (tell de IA).
- **Marca dos capas:** "IA aplicada" (descriptor) + "Ingeniería pausada con IA" (firma).
- **Accesibilidad:** WCAG 2.2 AA. El teal `#3BAA8C` NO sirve para texto de cuerpo
  (contraste 2.55:1); usar `ink`/`ink-mute-strong` para texto, teal solo acento.
- **Honestidad:** prohibido usar el caso "80.000€" (no referenciable) y cualquier
  número no atribuible a Zanovix.
